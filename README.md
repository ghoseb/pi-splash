# pi-splash

Splash screen for [pi](https://github.com/badlogic/pi-mono) with gradient logo and motivational taglines.

## Installation

```bash
pi install git:github.com/ghoseb/pi-splash
```

## Features

- Gradient pi logo (pink-to-cyan)
- Motivational taglines
- Keyboard shortcuts reference
- Auto-dismisses on first user message
